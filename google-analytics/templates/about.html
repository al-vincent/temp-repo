About this site

This website doesn't really do very much; it mainly exists for experiments with Google Analytics (e.g. tracking visits, click-throughs etc).


It has a few placeholders, just to differentiate between pages; a graph, a map and a Bootstrap carousel. But that's about it really.


For (minor) interest; the server-side code is built in Python, using the Flask microframework, and deployed using Heroku. Styling is done with Bootstrap. Charts are built using Chart.js, and maps are built using Leaflet.js.
